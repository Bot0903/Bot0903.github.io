NecroSpell

This is a top down shooter where players will have access to spells and fight against waves of zombies. This game is still under development so I still plan on adding new features to the game like more spells types, spell combos, enemy types, and a art direction

Notable elements

- Controller Support
- Scriptable Objects
- Menus

Controller Support

Controller support was something I have wanted to learn and so in this game I implemented controller support for both the menus and also the gameplay. The new input system is something that I am still learning so there are some parts that are a bit buggy or have odd workarounds but I hope to improve as I work more on this game.

Scriptable Objects

Scriptable Objects allow for easy creation of items or objects that have similar properties and attributes. This is a very useful tool for efficiency, scriptable objects can be used in many situations so I look forward to improving my use of scriptable objects.

Menus

For this game I have created a fully functioning main menu followed by a pause menu all controller supported. Menus are still something I struggle with but I have learned a lot while working on this project.
