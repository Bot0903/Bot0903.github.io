Platformer Demo

This platformer demo was made with many different platforming game elements and was made in a way where it can serve as a starting point for other possible platforming projects. This project was mainly made to display my abilities with 2D Unity elements.

Notable elements of this project

- 2D Animation
- Player Movement
- Simple Combat
- Parallax Background

2D Animation

This game demonstrates the use of 2d animations, the character along with the animation are sourced from the Unity store. I however had to set up the animations to fit with my game. Animation events and properties are used to trigger hit registration for combat and also used for triggering combo hit systems.

Player Movement

As a platformer demo the player movement is one of the main points of this build. It features a basic movement system but also features wall jumping and a variable jump mechanic.

Simple Combat

A simple combat system is also added into this game with a working enemy and simple AI. Player attacks with a combo system, consecutive attacks will chain together with an animation. Enemy AI will search out the player and attack when in range.

Parallax Background

The background was created with a parallax effect, and also loops when the player moves far enough away. This parallax effect can be tweaked to fit other layered backgrounds.
