The Four Gods

This web based game was one of the first games that I made when learning JavaScript. Its art design was mostly an afterthought thus the crazy colours and marquee text. The main feature of this game was the use of java script to create a grid map that is fully traversable and an inventory system that can add and remove items.

Notable elements of this project

- Grid Map
- Inventory / Items

Grid Map

The grid map is done through a nested array creating a 3x3 grid with an extra space on the bottom, making this map a total of 10 traversable spaces. Each map space has its own flavour text and intractable element which is stored in a switch case and checked for if the player chooses to interact.

Inventory / Items

The inventory for this game is quite simplistic using an string array to store item names, then using push and splice to add and remove items respectively. This game of course has items that can be dropped and picked up. In order to be able to have items sit on spaces of the map a second nested array is set up in tandem with the map array, this array only records the location of items dropped on the ground whether that be from the player dropping something or the game. This way when the game checks where the player is it can also check if there are item(s) on that space and display it.
